# LOMO Route Studio Support

LOMO Route Studio helps you turn selected iPhone photos into a short route-story video using on-device processing.

## Support topics

- Photo library permission issues
- GPS-tagged photo availability
- Route-story generation problems
- Save to Photos issues

## Before contacting support

- Update to the latest build
- Make sure the selected photos include GPS metadata when possible
- Confirm the app has photo library access
- Try creating a shorter route with fewer photos if the device is under heavy load
