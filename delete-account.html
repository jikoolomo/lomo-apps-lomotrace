Lomo Trace 계정 및 데이터 삭제 안내

Lomo Trace는 사용자의 핵심 기록 데이터를 중앙 서버가 아니라 사용자 기기 내부에 로컬 저장하는 오프라인 중심 앱입니다. 따라서 계정 또는 프로필 삭제와 데이터 삭제는 주로 앱 내부에서 직접 수행됩니다.

중요: Lomo Trace에서 삭제되는 것은 앱 내부의 로컬 프로필 및 로컬 데이터입니다. 이것은 Google 계정 자체 또는 Apple ID 자체 삭제를 의미하지 않습니다.

앱 내 삭제 방법

  1. Lomo Trace 앱을 실행합니다.
  2. 설정(Settings) 또는 프로필 관련 메뉴로 이동합니다.
  3. 계정 탈퇴 / 프로필 삭제 / 데이터 삭제 항목을 선택합니다.
  4. 안내 문구를 확인한 뒤 삭제를 진행합니다.

삭제 시 제거되는 항목

  • 해당 로컬 프로필
  • 핀 및 여행 기록
  • 컬렉션
  • 앱 내 연결된 사진 정보
  • 캐시 데이터
  • 관련 로컬 디렉토리 및 파일
  • 앱 내부에서 관리되는 관련 메타데이터

삭제 방식에 대한 안내

Lomo Trace는 사용자의 사적인 여행 기록을 저장하는 중앙 서버 데이터베이스를 운영하지 않습니다. 따라서 삭제는 일반적으로 기기 내부 데이터 제거를 의미합니다.

삭제는 영구적일 수 있으며, 삭제된 로컬 기록은 복구되지 않을 수 있습니다. 기기 분실, 앱 삭제, 저장소 손상, 초기화 등의 경우에도 개발자는 일반적으로 로컬 데이터를 복구할 수 없습니다.

외부 로그인 계정 관련 안내

Lomo Trace 로컬 프로필을 삭제하더라도 다음 계정 자체는 삭제되지 않습니다.

  • Google 계정
  • Apple ID

위 계정들은 각 제공자의 정책과 서비스에 따라 별도로 관리됩니다.
